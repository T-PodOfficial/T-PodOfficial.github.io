Features

Easy to Use

T-Pod's control using 0.91" inch display a 5 buttons is very easy.

Very powerfull

T-Pod uses Peltier's element for heating and cooling and two resistance wires for cooling.

Ecological

T-Pod is made from recycled parts and old components. Constuction is made out of wood and reinforced with iron.

High computing power

T-Pod is powered by raspberry pi 4B 4GB RAM and one NodeMCU.

For any mug

You can put almost any mug in to T-Pod.

Built-in google assistant

You can connect T-Pod with google assistant. Then you can use it as smart home and you can control T-Pod using voice and google assistant

Open-Source

Project T-Pod is open-source, so every source-codes are on github.

Mobile settings

You can set more things on your mobile phone or on any devices that has wifi connectivity.
